## 1. 核心概念(修订)

夜晚花园,萤火虫睡在家花上。孩子的手伸到传感器前——萤火虫**立刻醒来飞到屏幕中央贴着
"玻璃"跳舞**(绕圈盘旋),手离传感器越近,舞越快、绕圈越大、音调越高;手拿开,它挥挥手
飞回花上睡觉。玩的时候随手一挥——萤火虫就朝**同一个方向**翻个跟头(往右挥=往右翻),
推近=它凑到玻璃前变大,拉远=害羞缩小,画圈=它翻个筋斗云。自解释性:**手进=它醒,手出=
它睡,手近=它疯,手挥=它翻**——四条因果全部 <100ms、全部方向/强度一致,零词汇量。

## 3. 状态机(v2.1)

```
[NO_UNIT] ──(Gesture init 成功,手势模式)──► [SEEK]
    ▲(拔线/连续读失败 ERR_STREAK_LOST)──────┘

[SEEK]  萤火虫睡家花/慢眨眼 ──(在场信号 ON)──► [DANCE]
[DANCE] 贴玻璃盘旋(强度三档)+ 手势→方向翻滚 + 近距音阶
        ──(在场信号 OFF 且保持期 PRES_HOLD_MS 耗尽)──► 挥手再见 → 回家动画 ──► [SEEK]
```

- 在场信号 ON/OFF 用**迟滞 + 保持**(§4),DANCE 内短暂检不到手不掉状态。
- 翻滚是 DANCE 内的瞬时动画(~220ms),不是子状态;同方向 `TUMBLE_COOLDOWN_MS` 冷却,
  不同方向随时打断(`lv_anim_delete` 配对)。
- 省电正交同 v2:非 AWAKE 冻结;NAP 中在场/手势可唤醒(唤醒帧不判定);DEEP 断 5V 唤醒后
  重发手势模式初始化。

## 4. 传感层:在场信号 + 手势事件

### 4.1 在场信号(0xB0 亮度为主,0xB1-0xB2 尺寸备用)

- 读取:每帧(33ms)bank0 读 0xB0(1 字节)——沿用"写寄存器号+STOP、单独读"两笔事务。
  0xB1-0xB2(尺寸,12 位)P1 一并读出进标定日志,若亮度语义不好用再切尺寸做主信号。
- ⚠️ **语义待实机标定**(v1 README 已警告:默认手势初始化表下这些寄存器的读数语义
  datasheet 没背书)——P1 第一件事就是拿标定日志看真实读数分布(无手/手远/手近),
  阈值全部占位、可调。
- **平滑与判定**:EMA(`PRES_EMA_PCT` 30%)→ 迟滞开关(`PRES_ON_TH`/`PRES_OFF_TH`,
  占位 20/10)→ OFF 后保持 `PRES_HOLD_MS`(1500ms)才真正判离场(桥接实测 2.5s 级中断的
  主体;保持期内舞照跳,只降到最低档)。
- **强度三档**:平滑值按 `PRES_LVL2_TH`/`PRES_LVL3_TH`(占位 60/140)分 远/中/近,驱动
  舞速三档 + 音阶三档(§5.1)。档位切换加 `PRES_LVL_HYST` 迟滞防蹦档。

### 4.2 手势事件(沿用 v1 已验证路径)

0x43/0x44 手势结果,DFRobot 位映射(M5Stack U127 官方示例同款,v1 已核实);九种手势全部
消费但**零必修**:

| 手势 | 萤火虫反应(方向全等) |
|---|---|
| LEFT/RIGHT/UP/DOWN | 朝同方向快速翻滚一小段(~220ms)再回盘旋轨道 |
| FORWARD | 凑到玻璃前放大一档(快进快出) |
| BACKWARD | 害羞缩小 + 顿 + 弹回(呼应躲猫手感) |
| CW / CCW | 顺/逆时针筋斗云一整圈(8 方位查表) |
| WAVE | 左右欢摆 + 咯咯笑音 |

每种配 ≤4 音短音签 + `HAPTIC_BUMP_LIGHT`(WAVE 用 `HAPTIC_COLLECT`),同方向冷却
`TUMBLE_COOLDOWN_MS`(400ms)。方向映射若实机"反了",改 `unit_gesture_read()` switch
(v1 README 既有预案)。

## 5. 表现层

### 5.1 盘旋舞 + 近距音阶

- **盘旋**:8 方位查表(固定 `OCT_DX/DY`,非逐帧三角函数)绕屏幕中心离散步进,步进间隔
  按强度三档 `DANCE_STEP_MS_L1/L2/L3`(140/100/70ms),半径两档(远=小圈、中近=大圈)。
- **近距音阶**:强度档**上跳**播升音({523→659→784} 对应档)、**下跳**播降音——
  "手越近音越高"的方向一致听觉映射(替代 v2 的 Y 轴竖琴);单音 ~40ms 低幅,换档才响,
  不连刮。
- 「你好」时刻(离场 ≥2s 后再入场):问候双音 + `HAPTIC_HELLO`;「再见」:回家前欢摆
  一下 + 降双音(无震)。

### 5.2 夜花园目标与进度(P2,机制随 v2.1 重定义)

5 目标内容表沿用 v2 §5(粉花苞/蓝铃花/瞌睡鸟/青蛙/星星草,音签/触觉/灯带列全部照旧),
但**唤醒机制**从位置 dwell 改为:
- **方向拜访**:DANCE 中朝某方向挥手,萤火虫翻滚过去时若该方向有沉睡目标 → 目标预亮
  (即时反馈"你指到它了");同一目标被拜访 `VISIT_WAKE_N`(2)次 → 苏醒 BLOOM 全通道。
- **在场兜底**:累计在场每 `PRES_WAKE_FALLBACK_MS`(45s)自动唤醒一个"最久没动静"的目标
  ——保证纯乱挥也能走到 5/5(零失败:进度永远可达)。
- 5/5 → 天亮派对 → 暮色重置(v2 §5.3 原文照旧);月亮彩蛋改为 CW/CCW 筋斗云时随机
  小概率触发流星(不占进度)。

### 5.3 魔法灯笼(P4,v2 §5.4 原文有效)

在场 ON 亮起、强度三档亮度、BLOOM 闪对应色、SEEK 慢呼吸——语义与新传感层直接对齐。
